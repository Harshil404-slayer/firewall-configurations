# 🔐 Firewall Configuration Project (Linux & Windows)

## 👤 Author
**Harshil Patel**

## 📌 Overview
This project demonstrates basic firewall configuration and management using **UFW (Linux)** and **Windows Defender Firewall**. It includes creating, testing, and removing firewall rules to control network traffic and improve system security.

---

## 🧰 Tools Used
- Linux (UFW - Uncomplicated Firewall)
- Windows Defender Firewall
- Terminal / Command Line
- Windows Firewall Advanced Security (wf.msc)

---

## ⚙️ Features / Tasks Performed

### Linux (UFW)
- Enable/Disable firewall
- Allow and deny specific ports (22, 23, 80, 443)
- Block IP addresses
- Allow port ranges
- Delete firewall rules
- View firewall status

### Windows Firewall
- Create inbound rules
- Block specific ports (e.g., Telnet 23)
- Allow SSH, HTTP, HTTPS ports
- Block specific IP addresses
- Remove firewall rules

---

## 🧪 Testing
- Verified blocked ports using telnet command
- Checked rule application using status commands (Linux)
- Confirmed inbound rule behavior in Windows Firewall
